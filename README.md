# Skills

这个仓库收集可复用的 Codex skills。

## Skills

- [中文 DOCX 编辑](./chinese-docx-editing/SKILL.md)
- [Markdown Quiz HTML Generator](./markdown-quiz-html/SKILL.md)

## 使用方式

- 在 Codex 中调用 ` $chinese-docx-editing ` 处理中文 Word 文档
- 在 Codex 中调用 ` $markdown-quiz-html ` 把已审核的 markdown 题库生成刷题 HTML 和答案文件
- 适合需要保留排版、题目顺序和离线刷题体验的题库整理任务
